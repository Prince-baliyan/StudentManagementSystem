# 🖥️ PyQt5 GUI Application – Multi-Page Dashboard System

This project is a desktop application built using **Python and PyQt5**. It provides a modern graphical user interface with multiple pages, including login, registration, and dashboard management.

---

## 🚀 Project Overview

This application demonstrates how to build a **multi-window GUI system** using PyQt5 with structured layouts and navigation.

It includes:

* User authentication UI (Login/Register)
* Dashboard with multiple pages
* Form-based data entry
* Table display interface
* Sidebar navigation system

---

## ✨ Features

* 🔐 **Login & Registration System**

  * Input fields for user credentials
  * Navigation between login and signup pages

* 📊 **Dashboard Interface**

  * Multi-page layout using `QStackedWidget`
  * Organized UI sections

* 🧭 **Sidebar Navigation**

  * Easy switching between pages
  * Clean layout design

* 📝 **Forms & Input Fields**

  * Multiple input fields for data handling
  * Structured form UI

* 📋 **Table View**

  * Data display using `QTableWidget`

* 🎨 **Modern UI Design**

  * Gradient backgrounds
  * Styled buttons and layouts

---

## 🛠️ Tech Stack

* **Language:** Python 3
* **Framework:** PyQt5
* **UI Design:** Qt Designer (converted to Python using pyuic5)

---

## ▶️ How to Run

### 1. Install Dependencies

```
pip install PyQt5
```

### 2. Run the Application

```
python designui2.py
```

---

## 🧠 How It Works

* The application uses **QStackedWidget** to switch between different pages
* UI components like buttons and inputs are arranged using layouts
* Navigation is controlled by switching indexes of stacked widgets
